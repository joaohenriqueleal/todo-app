# TASKEN TODO-APP

Esta aplicação apresenta um gerenciador de tarefas diárias, com gráfico de evoluções, e aumentos de níveis pessoais estilo RPG.
Com sistema de Login e Registro em LocalStorage e SeessionStorage, deleção e edição de usuários e tarefas, e muito mais!

# TECNOLOGIAS ULTILIZADAS

- HTML
- CSS
- JavaScript
- React (JS)
- Tailwind
- ChartJS

# OQUE APRENDI DURANTE O DESENVOLVIMENTO?

- Deploy de projetos React de forma segura.
- Manipulação de estados complexos e hierarquia de componentes.
- Layout responsivo, intuitivo e de fácil leitura.
- Modularização de componentes React e Tailwind.
